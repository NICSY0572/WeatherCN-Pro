# WeatherCN-Pro

v0.3.0 修正版。

重要：
- `WeatherCN.plugin` 是 Loon 插件配置文件。
- `weathercn-pro.js` 是真正执行的 JavaScript 脚本。
- 不要把 `WeatherCN.plugin` 当作本地脚本运行，否则会出现 `Invalid character: '#'`。

Loon 订阅地址：

```text
https://raw.githubusercontent.com/NICSY0572/WeatherCN-Pro/main/WeatherCN.plugin
```

验证日志：

```text
[WeatherCN Pro 0.3.0] matched url=...
[WeatherCN Pro 0.3.0] inject success
```
